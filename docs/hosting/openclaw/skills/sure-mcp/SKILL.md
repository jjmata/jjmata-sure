# Skill: sure-mcp

Use this skill for questions that require pulling financial data from Sure MCP tools.

## When to Use
- "How much did I spend on groceries this month?"
- "Show my biggest subscriptions."
- "What changed in my cash flow since last month?"

## Workflow
1. Identify the user intent and needed dimensions (time window, account set, categories).
2. Fetch relevant data with MCP tools.
3. Reconcile totals before reporting (avoid mismatched sums).
4. Present findings with:
   - one-sentence headline,
   - bullet breakdown,
   - optional next action.

## Response Template
- **Headline**: one sentence with the key number.
- **Breakdown**: 3-6 bullets with supporting facts.
- **Action**: one practical suggestion.

## Quality Checklist
- Numbers tie out.
- Time range is explicit.
- Any inferred assumption is clearly marked.
